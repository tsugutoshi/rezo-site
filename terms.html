Rezo 利用規約

最終更新日: 2026年2月8日

第1条（適用）

  1. 本利用規約（以下「本規約」）は、Rezo（以下「本アプリ」）の利用に関する条件を定めるものです。
  2. ユーザーは、本アプリを利用することにより、本規約に同意したものとみなされます。

第2条（定義）

本規約において、以下の用語は次の意味で使用します。

  • 「本サービス」: 本アプリが提供するすべての機能（在庫管理、レシピ提案、買い物リスト、食材スキャン、AI解析等）
  • 「ユーザー」: 本アプリを利用するすべての個人
  • 「コンテンツ」: ユーザーが本アプリを通じてアップロード、送信、または保存するデータ（画像、テキスト等）

第3条（アカウント）

  1. 本アプリの一部機能を利用するにはアカウント登録が必要です。
  2. ユーザーは、登録情報を正確かつ最新に保つ責任を負います。
  3. アカウントの管理はユーザー自身の責任で行い、第三者による不正利用が発生した場合、速やかに当方へ通知してください。
  4. Google アカウントを利用したログインが可能です。Google アカウントの利用には Google の利用規約が適用されます。

第4条（提供するサービス）

本アプリは以下の機能を提供します。

  1. 在庫管理: 食材の登録・編集・消費管理
  2. AI食材認識: カメラまたはギャラリーから画像をスキャンし、AIが食材を認識
  3. レシート読み取り: レシート画像から食材を自動抽出
  4. レシピ提案: 在庫食材に基づくAIレシピ提案・栄養バランス表示
  5. 買い物リスト: 不足食材の自動提案・リスト管理
  6. 音声入力: 音声による食材の登録
  7. プッシュ通知: 消費期限のリマインダー通知
  8. AI学習データ貢献: ユーザーの同意のもと、食材画像をAIモデル改善に活用（オプトイン方式）

第5条（AI機能に関する免責）

  1. 本アプリのAI機能は、Google Gemini API および独自カスタムモデルを活用しています。
  2. AI機能の出力は参考情報であり、正確性・完全性を保証するものではありません。
  3. 食品アレルギー・食事制限のある方は、AI認識結果を鵜呑みにせず、必ず実際の食品表示をご確認ください。
  4. AI機能の精度はモデルのバージョンや画像の品質により変動します。
  5. 栄養計算の結果は概算値であり、医療的な判断の根拠としないでください。

第6条（禁止事項）

ユーザーは、以下の行為を行ってはなりません。

  1. 本アプリの不正利用またはリバースエンジニアリング
  2. 本アプリのサーバーまたはネットワークに対する妨害行為
  3. 他のユーザーのアカウントへの不正アクセス
  4. 法令または公序良俗に反する目的での利用
  5. 食材以外の画像を意図的にトレーニングデータとして送信する行為
  6. 本アプリの API を不正に大量呼び出しする行為

第7条（広告の表示）

  1. 本アプリには Google AdMob による広告が表示される場合があります。
  2. 広告の表示に関しては、Google の広告ポリシーが適用されます。

第8条（知的財産権）

  1. 本アプリおよびそのコンテンツに関する知的財産権は当方に帰属します。
  2. ユーザーがアップロードしたコンテンツの著作権はユーザーに帰属しますが、AI学習データ貢献に同意した画像については、モデル改善目的での利用権を当方に許諾するものとします。

第9条（サービスの変更・中断・終了）

  1. 当方は、事前の通知なくサービス内容の変更、中断、または終了を行うことができます。
  2. メンテナンスや不可抗力によるサービスの一時停止について、当方は責任を負いません。

第10条（免責事項）

  1. 本アプリの利用により生じたいかなる損害についても、当方は一切の責任を負いません。
  2. 本アプリは「現状有姿」で提供され、特定目的への適合性を保証するものではありません。
  3. Firebase、Google Cloud、その他第三者サービスの障害に起因する損害について、当方は責任を負いません。

第11条（規約の変更）

  1. 当方は、必要に応じて本規約を変更できます。
  2. 変更後の規約は、本アプリ内またはウェブサイトでの掲示をもって効力を生じます。
  3. 変更後に本アプリを利用した場合、変更後の規約に同意したものとみなします。

  1. 本規約は日本法に準拠し、解釈されます。
