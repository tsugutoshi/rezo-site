Rezo プライバシーポリシー

最終更新日: 2026年2月8日

はじめに

Rezo（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように利用・保護するかを説明するものです。

1. 収集する情報

1.1 アカウント情報

  • メールアドレス・パスワード: Firebase Authentication によるアカウント認証に使用
  • Google アカウント情報: Google サインイン利用時に、表示名・メールアドレス・プロフィール画像を取得
  • ユーザーID: Firebase が自動生成する匿名識別子

1.2 ユーザーが入力するデータ

  • 在庫データ: 食材名、数量、単位、カテゴリ、消費期限、保存場所
  • 買い物リスト: 購入予定の食材リスト
  • 調理履歴: レシピの閲覧・調理記録
  • プロフィール情報: ニックネーム、食事モード設定

1.3 画像データ

  • スキャン画像: カメラまたはギャラリーから取得した食材・レシート・パッケージの画像
  • AI学習用画像: ユーザーが「AIの学習に貢献」に同意した場合のみ、Firebase Storage に保存

1.4 自動的に収集する情報

  • デバイス情報: OS バージョン、デバイスモデル（クラッシュレポート・互換性の目的）
  • 利用状況: アプリの起動・機能利用の統計情報（Firebase Analytics が有効な場合）
  • プッシュ通知トークン: Firebase Cloud Messaging による消費期限通知の送信に使用

1.5 広告に関する情報

  • 広告識別子: Google AdMob がパーソナライズ広告の表示のために収集する場合があります
  • 広告に関するデータ収集は Google のプライバシーポリシーに準じます

収集した情報は、以下の目的でのみ利用します。

3. 第三者サービスとのデータ共有

本アプリは以下の第三者サービスを利用しており、それぞれのプライバシーポリシーが適用されます。

サービス用途共有されるデータ
Firebase Authenticationユーザー認証メールアドレス、認証トークン
Cloud Firestoreデータ保存在庫・買い物リスト等のユーザーデータ
Firebase Storage画像保存AI学習用画像（同意済みのもののみ）
Firebase Cloud Messagingプッシュ通知デバイストークン
Google Gemini APIAI食材認識・レシピ提案スキャン画像（Base64）、テキストプロンプト

重要: スキャン画像は AI 解析のために一時的に送信されますが、Google Gemini API 側でのデータ保持ポリシーは Google の利用規約に準じます。

4. AI学習データの取り扱い

4.1 オプトイン方式

AI学習データの収集は、ユーザーがスキャン結果確認時に「AIの学習に貢献」トグルを ON にした場合のみ行われます。トグルは各スキャン時に選択でき、いつでも OFF にできます。

4.2 保存されるデータ

  • 食材画像（リサイズ済み、最大1920×1920px）
  • 食材ラベル（名前、カテゴリ、ユーザー修正の有無）
  • 撮影元（カメラ/ギャラリー）
  • タイムスタンプ
  • ユーザーID（画像とアカウントの紐づけ管理用）

4.3 利用範囲

収集したデータは、本アプリの食材認識AIの訓練にのみ使用します。第三者への販売・提供は行いません。

4.4 削除権

ユーザーは、モデル管理画面の「投稿画像」タブからいつでも自分が投稿した学習用画像を削除できます。アカウント削除時には、当該ユーザーの学習用画像もすべて削除されます。

5. データの保存と管理

5.1 保存場所

ユーザーデータは Google Cloud（Firebase）の asia-northeast1（東京）リージョンに保存されます。

5.2 セキュリティ

  • Firebase Security Rules により、各ユーザーは自身のデータにのみアクセスできます。
  • 通信は TLS/SSL で暗号化されています。
  • パスワードは Firebase Authentication によりハッシュ化されて保存されます。

5.3 保存期間

  • アカウントが有効な間、ユーザーデータは保持されます。
  • アカウント削除後、ユーザーデータは30日以内に完全削除されます。

6. ユーザーの権利

  1. アクセス権: 自身のデータの閲覧（アプリ内の各画面から確認可能）
  2. 修正権: 在庫データ・プロフィール情報の編集
  3. 削除権: アカウントおよび関連データの削除（プロフィール画面から実行可能）
  4. AI学習データの削除: 投稿した学習用画像の個別削除
  5. オプトアウト権: AI学習データ貢献の拒否（トグルを OFF にする）
  6. 通知の停止: 端末の設定からプッシュ通知を無効化

7. 子どもの利用

本アプリは13歳未満の子どもを対象としていません。13歳未満の方が個人情報を提供していることが判明した場合、速やかに当該データを削除します。

8. カメラ・マイク・ストレージの権限

権限用途必須/任意
カメラ食材・レシートの撮影任意
マイク音声入力による食材登録任意
写真ライブラリギャラリーからの画像選択任意
通知消費期限リマインダー任意

権限を付与しない場合、該当機能は利用できませんが、他の機能には影響しません。

9. Cookie とトラッキング

本アプリのモバイル版では Cookie を使用しません。

10. ポリシーの変更

本プライバシーポリシーは、法令の変更やサービスの更新に伴い変更される場合があります。重要な変更がある場合は、アプリ内通知またはメールでお知らせします。
